# DRIFTANALYSE---Buckelwal-Timmy---Unabh-ngige-physikalisch-ozeanogr.-Untersuchung-Stand-Mai-2026
Methodik: Vorwärtsmodell ohne Rückwärtskalibrierung - Diese Studie rekonstruiert die Driftbahn des Buckelwals „Timmy“ im Zeitraum 2. bis 14. Mai 2026. Alle Modellparameter werden a priori aus Fachliteratur und Reanalysedaten abgeleitet; das Modell sagt eine Endposition voraus und vergleicht diese anschließend mit dem Fundort Anholt.
